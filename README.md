# ai-scam-detector
AI-powered scam detection for emails and messages. Paste suspicious content and get instant risk analysis.
🛡️ Student Scam Shield AI
Student Scam Shield is an AI-powered web application that helps students detect scam emails, phishing messages, and manipulation attempts.
The system analyzes suspicious text or message screenshots and identifies potential scam patterns such as urgency, fear tactics, and fake authority.
The goal of this project is to protect students from online fraud by providing instant scam risk analysis.
📌 Project Type
AI-powered text and image scam detection system
Features include:
Scam risk percentage analysis
Detection of manipulation tactics (Urgency, Fear, Authority, Greed)
Explanation of why the message may be suspicious
Ability to analyze both text messages and screenshots of emails/messages
🧠 AI Model
The system uses Natural Language Processing (NLP) and AI models to analyze the content of messages.
Possible detection techniques:
Keyword and pattern analysis
Language manipulation detection
AI-based classification of scam vs legitimate messages

🔎 How It Works
1️⃣ User pastes a suspicious email or message or uploads a screenshot.
2️⃣ The website sends the content to the backend API.
3️⃣ The AI model analyzes the message and checks for scam indicators such as:
Urgency
Threats
Fake authority
Financial requests
Suspicious links
4️⃣ The system returns:
Scam Risk Score
Detected manipulation tactics
Short explanation
📊 Use Cases
Students verifying suspicious scholarship emails
Checking internship or job offer messages
Detecting phishing attempts in emails
Identifying manipulation tactics in online messages
🎯 Goal of the Project
The goal of Student Scam Shield is to raise awareness and provide a simple AI tool that helps students identify and avoid online scams quickly and easily.
