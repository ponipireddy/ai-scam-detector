# Student Scam Shield AI

Student Scam Shield AI is a web-based system designed to help students detect scam emails, phishing messages, and manipulative online communications.

## Project Overview
This project analyzes suspicious emails or messages and provides a risk score along with detected manipulation tactics such as urgency, fear, authority pressure, and financial requests.

## Project Type
AI-assisted scam detection and psychological manipulation analysis.

## Tech Stack
- Frontend: Lovable AI website builder
- Backend: Python Flask API
- AI Logic: Rule-based scam detection
- Deployment: Web application

## How It Works
1. User pastes an email or message into the analyzer.
2. The backend processes the text.
3. The system detects scam patterns such as urgency, threats, suspicious links, and financial requests.
4. A risk score and explanation are generated.

## Features
- Scam risk percentage
- Manipulation tactic detection
- Email and message analysis
- Simple and user-friendly interface

## Future Improvements
- Machine learning based scam detection
- Screenshot and image scam analysis
- Browser extension integration
